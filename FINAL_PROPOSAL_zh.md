# 通过通道调制负正则化实现选择感知的框监督

**版本**: v5-final | **评分**: 9.4/10 | **审稿人**: GPT-5.4（5轮）| **线程**: 019d9660-6470-76b3-b29d-bda8b8893f95

---

## 1. 问题锚点

框标注系统性地遗漏小目标。训练过程将遗漏的正样本当作负样本，导致分割偏向大结构。安全区（距所有标注较远的体素）压制了被标注者遗漏的真实小目标。

**没有任何现有方法基于估计的标注倾向性来控制安全区负监督强度。**

**非目标**：不是新的PU学习框架、不是新的骨干/架构、不是通用噪声标签校正、不是伪标签流程。

**约束**：~3,500 A100小时；LiTS（主数据集）；8周；目标MICCAI（主要），CVPR/ICCV（冲刺）。

**成功标准**：（1）Q1五分位Dice ≥ +5pt 优于均匀安全区；（2）消融证明通道负监督在scaffold之上有额外增益。

## 2. 技术差距

**没有方法基于估计的标注倾向性来控制安全区负监督强度。** 所有现有框监督方法在标注区域外施加均匀负压力。

| 方法 | 问题 |
|------|------|
| 3D-BoxSup (2020) | SCAR假设：均匀倾向性。对尺寸相关选择不成立。 |
| BoxInst (CVPR 2021) | 启发式投影，非标注过程推导 |
| BLPSeg (TIP 2023) | 涂鸦偏差，启发式，无形式化通道 |
| MonoBox (AAAI 2025) | 均匀安全区，无尺寸感知调制 |
| GeoCoBox (AAAI 2026) | 仅几何约束，均匀负监督 |
| LooBox (ACM MM 2025) | 点引导，无标注倾向性模型 |
| Bekker (2019), Gong (2021) | 偏置PU用于分类，无空间安全区结构 |

**差距**：没有方法将标注倾向性与安全区负监督强度关联。

## 3. 贡献

**核心贡献（单一主导）**：逐成分通道调制负监督——一个标量函数 g_θ（≤10参数）查询每个安全区连通成分尺寸处的估计标注概率。标注者可能标注到的地方给予强 α，可能遗漏的地方给予弱 α。

**预承诺定位**：如果 g_θ ≈ linear，贡献仍然成立（单调尺寸感知调制 > 均匀）。校准相关声明减弱。标题/摘要相应调整。

**主标题**（若 g_θ 显著优于 linear）：Selection-Aware Box Supervision via Channel-Modulated Negative Regularisation

**备选标题**（若 g_θ ≈ linear）：Size-Aware Negative Modulation for Box-Supervised 3D Segmentation

**显式非贡献**：IPW（训练稳定性细节），3D紧致/填充（scaffold，非新颖），保序回归（标准工具）。

## 4. 方法

### 4.1 复杂度预算

| 组件 | 状态 | 参数量 | 角色 |
|------|------|--------|------|
| nnUNet v2 (3D fullres) | 复用 | ~30M | 骨干网络 |
| g_θ: 1D 保序回归 | 新增 | ~10 | 通道模型 |
| 框损失项 | 新增 | 0 | 仅损失函数 |

有意排除：伪标签、EMA、对比学习、SAM、空间单调性、RL。

### 4.2 建模命题1：选择感知的负样本风险

**假设：**
- **A1（尺寸相关选择）**：标注概率仅取决于连通分量大小：g_θ(log s)，单调非递减。
- **A2（预热充分性）**：预热后，预测的连通分量充分逼近真实分量，使 g_θ 给出有意义的倾向性估计。
- **A3（校准）**：g_θ 校准良好（通过陡度内 ECE < 0.10 验证）。

**命题。** 在 A1–A3 下，u ∈ S 处的期望负损失分解为：
- g_θ(log m(C(u))) · ℓ_neg(u) · P(y*=0 | 被观测) — **假阳性项**（惩罚）
- (1 - g_θ(log m(C(u)))) · ℓ_neg(u) · P(y*=1 | 被遗漏) — **遗漏目标项**（保护）

设 α(u) = g_θ(log m(C(u))) 仅保留假阳性压制项，保护可能被遗漏的成分。

### 4.3 标注通道模型 g_θ：精确拟合协议

**Real-Box 假设（R1）：** 用于通道拟合的标注框必须反映真实的人工标注过程——与仅框标注扫描使用相同的标注者和相同的协议。如果像素标注扫描没有独立的框标注，则使用模拟回退（从GT掩码生成bbox，按反比于尺寸的概率随机丢弃）并作为显式限制说明。

**输入**：像素标注子集（31张LiTS扫描，同时具有真值掩码和框标注）。

**逐步协议：**
1. 对每张像素标注扫描，提取所有真值连通分量（scipy.ndimage.label），记录每个分量体素数 |C_k|。
2. 提取标注框 {B_1, ..., B_M}。
3. 计算代价矩阵：cost(k, j) = 1 - IoU(bbox(C_k), B_j)。
4. 匈牙利匹配（scipy.optimize.linear_sum_assignment）。
5. 匹配对且 IoU > 0.3 → y_k = 1（已标注）；其余 → y_k = 0（遗漏）。
6. 报告 **unmatched_rate** = count(y_k=0) / N 和 **ambiguous_rate** = count(匹配前多个IoU>0.3) / N。
7. 构造训练对：(x_k, y_k) = (log|C_k|, 标注状态)。
8. 拟合：`IsotonicRegression(y_min=0.01, y_max=0.99, out_of_bounds='clip').fit(x, y)`
9. 同时提取：ρ_min = P10(|C_k ∩ B_j| / |B_j|)，ρ_max = P95。

**交叉验证**：像素子集内3折CV。每折报告陡度内ECE（附录）。若任一折ECE > 0.10，增加校准数据。

### 4.4 通道调制负监督 (L_channel_neg) — 核心创新

**步骤1：安全区。**
S = {u : d_chebyshev(u, ∪_j B_j) > d_safe}，d_safe = 5 体素（默认）。

**步骤2：双阈值CC提取（每 K_cc = 50 步，分离梯度）。**
1. M_low = 𝟙[f_θ > τ_low] ∩ S，τ_low = 0.3
2. 3D连通分量：{C_1, ..., C_n}（scipy.ndimage.label，CPU，<10ms/patch）
3. 对每个 C_k：V_high = {u ∈ C_k : f_θ(u) > τ_high}，τ_high = 0.5
   - |V_high| > 0 → **成熟**：m(C_k) = Σ_{u ∈ C_k} f_θ(u)，查询 g_θ
   - |V_high| = 0 → **新生** → α_k = α_min

**步骤3：赋值规则。**
- C_k 触及 patch 面 → α_k = 1.0（保守：不完整成分）
- 新生（无高阈值体素）→ α_k = 0.05（最大保护）
- 内部成熟 → α_k = clamp(g_θ(log m(C_k)), 0.05, 1.0)
- u ∈ S 无所属成分 → α(u) = 1.0（均匀：无预测结构）

**步骤4：损失。**
L_channel_neg = mean_{u ∈ S} [ α(u) · (-log(1 - f_θ(u) + ε)) ]

**直觉：**
- 模型在安全区预测*大*的正样本簇 → g_θ(大) ≈ 1 → 强负监督（"如果真实，标注者应该看到 → 可能假阳性"）
- 模型在安全区预测*小*的正样本簇 → g_θ(小)低 → 弱负监督（"可能是通道遗漏 → 保护"）

**低覆盖回退（确定性）：** 若 epoch 100 时 coverage_ratio < 0.10 → d_safe = 3。记录为超参事件，无需人工干预。

### 4.5 框约束 Scaffold (L_scaffold)

L_scaffold = L_tight + β · L_fill。复用组件，非新颖。

**项1：3D紧致性 (L_tight)** — IPW加权。
L_tight(j) = (1/n_slices) Σ_s softplus(κ − m_s)

**项2：充填率区间 (L_fill)** — IPW加权。
L_fill(j) = softplus(ρ_min·|B_j| − m_j)/|B_j| + α·softplus(m_j − ρ_max·|B_j|)/|B_j|

**IPW加权**：自归一化：w̃_j = w_j · |S| / Σ w_j，w_j = min(1/g_θ(log|C_j|), w_max)。

### 4.6 完整损失

```
L_box = L_scaffold + γ · L_channel_neg
L_total = λ_pix · (Dice + CE) + λ_box · L_box
```

### 4.7 诊断（每50步记录）

- **coverage_ratio** = Σ_{α<1} f_θ / Σ_S f_θ — 安全区中接受非均匀 α 的质量比例。若 < 0.10 → 触发 d_safe 回退。
- **nascent_ratio** = count_nascent / count_total — 新生成分比例。若 > 0.5 → 增加 τ_low。
- **mean_alpha** — 平均逐成分权重趋势。

### 4.8 训练协议

| 阶段 | Epoch | 数据 | 备注 |
|------|-------|------|------|
| 预拟合 | 0 | 像素子集 | g_θ（匈牙利协议），ρ_min, ρ_max。冻结。 |
| 热身 | 1–50 | 仅像素 | Dice+CE。 |
| 爬升 | 51–100 | 像素+框 | L_scaffold。L_channel_neg 从 epoch 60 启用。 |
| 完整 | 101–300 | 像素+框 | 所有项。CC每50步提取。 |

批次循环：[pixel, pixel, box]。无EMA。无重新校准。无伪标签。

### 4.9 推理

仅使用 f_θ。零开销。推理时不需要 g_θ。

### 4.10 失败模式

| 失败 | 检测 | 回退 |
|------|------|------|
| g_θ 校准失败 | ECE > 0.10 | 更多像素扫描或模拟回退 |
| 通道负监督 → FN | Q1召回率下降 | ↓ γ, ↑ α_min |
| 新生成分主导 | nascent_ratio > 0.5 | ↑ τ_low |
| 低覆盖 | epoch 100 时 coverage_ratio < 0.10 | d_safe = 3 |
| g_θ ≈ linear | 消融对比 | 降低校准声明；单调调制仍新颖 |
| 框非真实标注 | 数据集元数据检查 | 模拟回退 + 限制说明 |

## 5. 验证方案

### 5.1 核心实验：4配置对比

| 配置 | 描述 |
|------|------|
| C1: 仅像素 | 下界（31张扫描，Dice+CE） |
| C2: scaffold + 均匀负监督 | 基线（L_tight + L_fill + 均匀 α=1.0 安全区） |
| C3: scaffold + 线性负监督 | 启发式基线（α = clamp(a + b·log m)） |

**主要指标**：Q1五分位Dice（最小20%肿瘤按体积）

## 7. 实验交接

- **必须证明**：通道负监督(C4) > 均匀(C2)的Q1五分位Dice（+5pt）。g_θ > linear（C4 > C3）。
- **必须验证**：3折CV ECE < 0.10。机制活跃（coverage_ratio > 0.10）。
- **主数据集**：LiTS（31像素 + 73框，27验证）
- **最高风险**：g_θ ≈ linear → 降级为单调调制论文（仍新颖）。

## 8. 声明条件化

**若 g_θ 显著优于 linear**："选择感知校准通道调制" — 完整标题，完整声明。

**若 g_θ ≈ linear**："单调尺寸感知负调制" — 备选标题。贡献为逐成分通道调制负监督（机制无论如何新颖）。校准声明减弱。

**若机制不活跃（coverage_ratio ≈ 0）**：基本设计问题。排查 d_safe, τ_low, 训练动态。如实报告。
